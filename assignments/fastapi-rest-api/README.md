# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective

Learn how to build RESTful APIs using the FastAPI framework in Python. Students will create endpoints, handle HTTP requests, and return JSON responses, practicing modern web development skills.

## 📝 Tasks

### 🛠️ Create a Basic FastAPI App

#### Description
Set up a FastAPI application with at least one endpoint that returns a simple JSON response.

#### Requirements
Completed program should:
- Use FastAPI to create a web server
- Define at least one GET endpoint
- Return a JSON response

### 🛠️ Implement CRUD Endpoints

#### Description
Expand your FastAPI app to include endpoints for Create, Read, Update, and Delete operations on a resource (e.g., items, users).

#### Requirements
Completed program should:
- Provide endpoints for POST, GET, PUT, and DELETE
- Handle request data and return appropriate responses
- Use Python data structures to store resource data

### 🛠️ Add Input Validation

#### Description
Use FastAPI's validation features to ensure incoming data is correct and provide helpful error messages.

#### Requirements
Completed program should:
- Use Pydantic models for request validation
- Return error messages for invalid input
